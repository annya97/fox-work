Jēdziens Skaidrojums
@media noteikumi CSS tehnika, kas ļauj piemērot dažādus stilus dažāda lieluma un veida ierīcēm.
API Application Programming Interface - lietojumprogrammas saskarne. Tā ir rīku un protokolu kopa, kas ļauj sadarboties divām lietojumprogrammām.
CSS Cascading Style Sheets - kaskādes stila lapas. Tā ir valoda, kas apraksta HTML dokumenta stilu.
DOM Document Object Model - dokumenta objektu modelis. Tas ir veids, kā tiek attēlota tīmekļa lapa. DOM elementi ir, piemēram, div, p, html, body, span un citi.
HTML Hypertext Markup Language - hiperteksta iezīmēšanas valoda. Tā ļauj izveidot tīmekļa lapas attēlošanai jebkurā pārlūkprogrammā.
iframe Inline Frame Element - rindas rāmja elements. Tā ir HTML birka, kas ļauj attēlot iestarpinātu pārlūkošanas saturu.
JavaScript Augsta līmeņa programmēšanas valoda, ko izmanto, lai ieviestu interaktīvu saturu tīmekļa vietnēs.
LESS Leaner Style Sheets - vienkāršākas stila lapas. Atpakaļsaderīgs CSS valodas paplašinājums.
SASS Syntactically Awesome Style Sheets - sintaktiski satriecošas stila lapas. Tā ir preprocesora skriptēšanas valoda, kas tiek interpretēta vai kompilēta par CSS.
